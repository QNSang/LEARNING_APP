# IDEA - AI Learning App

Thu muc nay tong hop y tuong phat trien du an tu mot cong cu tao Knowledge Graph thanh mot he thong hoc tap thich nghi dua tren AI.

Muc tieu cua bo tai lieu:

- Dinh hinh san pham truoc khi code.
- Tong hop nhung pattern hay tu `neo4j-labs/llm-graph-builder`.
- Chuyen hoa logic GraphRAG/Knowledge Graph thanh logic phu hop voi ung dung hoc tap.
- De xuat data model, pipeline va roadmap phat trien.

## Huong di chinh

Huong phat trien nen la:

```text
Document Upload
-> Parse and Chunk
-> Knowledge Extraction
-> Learning Graph
-> GraphRAG Tutor
-> Quiz / Flashcard
-> Mastery Tracking
-> Spaced Repetition
-> Adaptive Learning Path
```

Khac biet quan trong: app khong chi "hoi dap voi tai lieu", ma giup nguoi hoc hieu, nho, kiem tra, phat hien lo hong va di theo lo trinh ca nhan hoa.
